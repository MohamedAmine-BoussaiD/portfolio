A clean, responsive personal portfolio website built with pure **HTML** and **CSS** .

## 🚀 Getting Started

No installation needed. Just open `index.html` in your browser.

```bash
# Clone or download the project, then:
open index.html
```

Or drag and drop `index.html` into any browser window.

---

## 📄 Sections

### Header
- Sticky dark navbar with smooth scroll navigation links
- Logo with cyan accent dot

### Hero
- Full-viewport dark section with animated dot-grid background
- Soft ambient glow effect (CSS only)
- Spinning conic-gradient ring around the profile photo
- Floating "AI 🤖" badge with CSS float animation
- Two CTA buttons — GitHub and LinkedIn

### Projects
- 4-column responsive grid
- Each card has a unique color identity (blue / teal / purple / orange)
- Image zoom on hover, colored overlay reveal
- Colored meta labels and matching hover button per card

### Certifications & Badges
- Auto-scrolling infinite carousel (CSS animation, no JS)
- Pauses on hover
- Glow effect on badge hover

### Footer
- Minimal dark footer
